Please write a Python program which has the following functionality in execution:

•Ask the user to enter a 4-digit positive integer.

•Using all the digits in the user input integer, find the smallest permutation so that the constructed integer is the smallest.

•Show the integer constructed by the smallest permutation of all the digits in the user input integer in the console.

•Please see the examples below to better understand the expected program behavior.

Example 1
User Input Integer 6328
Expected Program Output Value 2368
Explanation The digits in the user input integer are 6, 3, 2, and 8.  The smallest permutation of the digits is 2368.
Example 2
User Input Integer 3131
Expected Program Output Value 1133
Explanation The digits in the user input integer are 3, 1, 3, and 1.  The smallest permutation of the digits is 1133.
Example 3
User Input Integer 6060
Expected Program Output Value 66
Explanation The digits in the user input integer are 6, 0, 6, and 0.  The smallest permutation of the digits is 0066.  The integer constructed by this permutation is 66.

•Your can assume that the user will always provide valid input values.  You do not need to worry about invalid input or error handling.

•Your program will be tested (graded) using multiple sets of valid input values.  As long as the input values are valid, your program should always generate correct output values.

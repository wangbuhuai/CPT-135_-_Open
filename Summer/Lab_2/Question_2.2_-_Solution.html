Lab 2 - Question 2 (2 Points)

Some integers are stored in a list variable my_list.

•Please note that "integers" include positive and negative integers.

•Initially, you do not know the number of integers in my_list.

•my_list is guaranteed to contain at least 1 integer in it.  In other words, it is non-empty.

Please write one line of code to show the sum of the first integer and last integer in my_list in the console.

•Please see the examples below to better understand the expected program behavior.

Example 1
Value of my_list [1, 0, 2, −2]
Expected Output Value −1
Explanation The first integer is 1; the last integer is −2.  So, their sum is −1.
Example 2
Value of my_list [3]
Expected Output Value 6
Explanation The first integer is 3; the last integer is also 3 (actually 3 is both the first and last integer).  So, their sum is 6.

•Your can assume that the values stored in my_list are always valid.  You do not need to worry about invalid values or error handling.

•Your program will be tested (graded) using multiple sets of valid input values.  As long as the input values are valid, your program should always generate correct output values.

Solution
1 print(my_list[0] + my_list[-1])

Please write a Python program which has the following functionality in execution:

•Ask the user to enter a non-zero integer.

•Determine whether the user input integer is even or odd.

•Determine whether the user input integer is positive or negative.

•Show the results in the console.

•Please see the examples below to better understand the expected program behavior and output format.

Example 1
User Input Value 713
Expected Program Output Positive
Odd
Example 2
User Input Value −58
Expected Program Output Negative
Even

•Your can assume that the user will always provide valid input values.  You do not need to worry about invalid input or error handling.

•Your program will be tested (graded) using multiple sets of valid input values.  As long as the input values are valid, your program should always generate correct output values.

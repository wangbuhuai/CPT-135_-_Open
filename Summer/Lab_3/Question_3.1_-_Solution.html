As a gold sponsor of the Kansas City Royals, Papa John's provides discounts for local baseball fans.  The rules are below:

•If Royals won, online customers will enjoy 20% off for the entire order during the next day.

•If Royals won and scored 3 or more, online customers will enjoy 50% off for the entire order during the next day.  This rule will overwrite the previous rule if the conditions are satisfied.

Please write a Python program which has the following functionality in execution:

•Ask the user to enter the score of Royals last night.  Use a reasonable data type to store the user input value.

•Ask the user to enter the score of Royals' opponent last night.  Use a reasonable data type to store the user input value.

•Ask the user to enter the total amount of the Papa John's online order.  You do not need to worry about tax, tip, or other fees.  Use a reasonable data type to store the user input value.

•Based on the discount rules mentioned above, show the amount after discount (if any) in the console.

•Please see the examples below to better understand the expected program behavior.

Example 1
Score of Royals 4
Score of Opponent 6
Total of Online Order 22.50
Expected Output Value 22.50
Explanation Royals lost, so there will be no discount.
Example 2
Score of Royals 2
Score of Opponent 1
Total of Online Order 22.50
Expected Output Value 18.00
Explanation Royals won, but scored only 2 (less than 3), so 20% discount is applied.
Example 3
Score of Royals 5
Score of Opponent 4
Total of Online Order 22.50
Expected Output Value 11.25
Explanation Royals won and scored 5 (more than 3), so 50% discount is applied.

•Your can assume that the user will always provide valid input values.  You do not need to worry about invalid input or error handling.

•Your program will be tested (graded) using multiple sets of valid input values.  As long as the input values are valid, your program should always generate correct output values.

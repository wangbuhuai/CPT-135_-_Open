Please write a Python program to solve the following quantitative question.

The length of a piece of railway track is 80 feet (fixed).  The distance between two cities is miles (input value).  Therefore, how many pieces of railway track are needed to build a railroad between the two cities?

•Ask the user to enter the distance (in miles) between two cities, which is an integer.  Store the user input value in a variable.

•Please note that the program user cannot see or understand your code.  Therefore, you should provide sufficient screen prompts in the console to let the user clearly know what to input.

•Calculate the number of pieces of railway track needed to build a railroad between the two cities.  Is there any "pitfall" in this question?

•Show the calculation result (number of railway tracks) in the console, which should be an integer.

•Your can assume that the user will always provide valid input values.  You do not need to worry about invalid input or error handling.

•Unit conversion: 1 mile = 5,280 feet

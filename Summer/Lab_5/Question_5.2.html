A string is a palindrome if it is the same reading from left to right and from right to left.  For example, "I saw I was I" is palindromic.

Please write a function that takes a string as its only argument.  The function tests whether the string is palindromic or not.  The function returns True if the string is palindromic; False otherwise.

•The return type of the function should be bool (not str).

•Please see the examples below to better understand the expected function behavior.

Example 1
Function Argument "I saw I was I"
Expected Function Return Value True
Expected Function Return Type bool
Example 2
Function Argument "race a car"
Expected Function Return Value False
Expected Function Return Type bool

•Please only complete the required function.  Do not write a main() program or any testing code.

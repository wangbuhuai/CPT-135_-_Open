Please write a function that takes a string as its only argument.  The function removes all the whitespaces in the string.

•Whitespace characters include space (' '), newline ('\n'), and tab ('\t').

•Your function should change the input string.  Instead, it should return a copy of the input string with all whitespaces removed.

•The input string my not contain any whitespace characters.  If so, your function should do nothing (but cannot crash) and return a copy of the input string.

•Whitespace characters may appear anywhere (e.g., at the beginning, in the middle, and/or at the end) in the input string.  Your function should remove all of them.

•Please see the example below to better understand the expected function behavior.

Example
Function Argument " -- CPT-135 --\tProgramming in Python\n"
Expected Function Return Value "--CPT-135--ProgramminginPython"
Expected Function Return Type str

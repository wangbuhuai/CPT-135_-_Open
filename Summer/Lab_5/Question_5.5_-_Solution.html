Please write a function that takes a list of integers as its only argument.  The function finds and returns the length of the longest increasing segment in the list.

•You cannot assume (or guess) the size of the input list.  In other words, no matter how many integers are stored in the input list, your function should correctly find the length of the longest increasing segment in it.

•If the input list is a constant list (e.g., [4, 4, 4, 4, 4]) or decreasing list (e.g., [20, 15, 9, 2, −2, −18]), then your function should return 1 (not 0), since any element itself can be considered as an increasing segment of length 1.

•The input list is guaranteed to be non-empty but is not sorted in any way.

•Please see the example below to better understand the expected function behavior.

Example
Function Argument [1, 0, 2, 5, −3, 0, 0, 4, 8, 12, 10, 16, 14]
Expected Function Return Value 4
Expected Function Return Type int
Explanation

There are 4 increasing segments in the input list (shaded with different colors):

[1, 0, 2, 5, −3, 0, 0, 4, 8, 12, 10, 16, 14]

The longest increasing segment is 0, 4, 8, 12 which has length 4.

•Please only complete the required function.  Do not write a main() program or any testing code.

•(Hint) The algorithm of finding the longest increasing segment is very similar to the algorithm of finding the maximum element.  You need to keep a variable that stores the length of the longest increasing segment "seen so far".  Then, use a loop to iterate through the input list.  Every time when you find a longer increasing segment, you update the variable.  After iterating through the list, the variable's value will be the answer.  Good luckâœŒ!

26 		def longest_increasing_segment(li):
    """ Finds the length of the longest increasing segment in a list
        :param li: a list to find the length of its longest increasing segment
        :type li: List[int]
        :return: length of the longest increasing segment in the list
        :rtype: int
    """
    begin = 0  # Beginning index of current increasing segment
    end = 0  # End index of current increasing segment
    length = 1  # Length of current longest increasing segment
    for i in range(1, len(li) + 1):
        if i < len(li) and li[i] > li[i - 1]:
            # If current value is greater than the previous value,
            # then current increasing segment is growing.
            end = i  # 'i' is the new end index.
        else:
            # If current value is smaller than the previous value,
            # then a whole increasing segment is found.
            size = end - begin + 1
            # If the segment just found is longer than the longest seen so far,
            # then "length" needs to be updated.
            if size > length:
                length = size
            # Initialize the next increasing segment.
            begin = end = i
    return length

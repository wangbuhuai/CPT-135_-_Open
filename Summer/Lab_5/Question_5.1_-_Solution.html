Anagrams are words or phrase you spell by rearranging the letters of another word or phrase.  A few well-known anagrams are listed below for you to better understand the concept.

•"study" and "dusty"

•"eleven plus two" and "twelve plus one"

•"bad credit" and "debit card"

Please write a function that takes two strings, s and t, as arguments.  The function tests whether s and t are anagrams or not.  The function returns True if s and t are anagrams; False otherwise.

•The return type of the function should be bool (not str).

•Please see the examples below to better understand the expected function behavior.

Example 1
Function Arguments s = "study"
t = "dusty"
Expected Function Return Value True
Expected Function Return Type bool
Example 2
Function Arguments s = "car"
t = "rat"
Expected Function Return Value False
Expected Function Return Type bool

Lab 5 - Question 3 (4 Points)

The majority element in a list is the element that appears more than half times in the list.  For example, in list [5, 6, −3, 5, 5, 6, 5, 5, 3], element 5 appears 5 times, which is more than half of the size of the list (9).  Therefore, 5 is the majority element in this list.

Please write a function that takes a list of integers as its only argument.  The function finds and returns the majority element in the list.

•You cannot assume (or guess) the size of the input list.  In other words, no matter how many elements are stored in the list, your function should correctly find the majority element in it.

•Although not every list has a majority element, in this question, you can assume that a majority element always exists in the input list.

•Please see the examples below to better understand the expected function behavior.

Example 1
Function Argument [5, 6, −3, 5, 5, 6, 5, 5, 3]
Expected Function Return Value 5
Expected Function Return Type int
Explanation The size of the list is 9.  Element 5 appears 5 times in the list, which is more than half of the size of the list.  Therefore, the majority element is 5.
Example 2
Function Argument [0, 2, −2, 2, 2, 4, 3, 2, 2, 2, 2, 0]
Expected Function Return Value 2
Expected Function Return Type int
Explanation The size of the list is 12.  Element 2 appears 7 times in the list, which is more than half of the size of the list.  Therefore, the majority element is 2.

•Please only complete the required function.  Do not write a main() program or any testing code.

•(Hint) If the list is unsorted, then is the majority element always the first element?  Is the majority element always the last element?  Is the majority element always the middle element?  What if the list is sorted?

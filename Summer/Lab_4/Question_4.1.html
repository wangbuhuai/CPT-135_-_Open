Some integers are stored in a list variable my_list.  Please write one line of code to calculate and show the subtraction difference between the maximum value and minimum value in the list.  The result should be non-negative.

•my_list is guaranteed to be non-empty.

•Your program should write the result in the console.

•Please see the examples below to better understand the expected program behavior.

Example 1
Values in my_list [4, 2, −3, 6, 6, 7, 3]
Expected Program Output 10
Explanation The maximum value in the list is 7; the minimum value in the list is −3.  Their difference is 7 − (−3) = 10.
Example 2
Values in my_list [6, 6, 6]
Expected Program Output 0
Explanation The maximum value in the list is 6; the minimum value in the list is also 6.  Their difference is 6 − 6 = 0.

•Your can assume that the values stored in my_list are always valid.  You do not need to worry about invalid values or error handling.

•Your program will be tested (graded) using multiple sets of valid input values.  As long as the input values are valid, your program should always generate correct output values.

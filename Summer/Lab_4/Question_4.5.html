The median of a non-empty sorted list of integers is defined as below:

•If the size of the list is an odd number, then the median of the list is just the middle value.

•If the size of the list is an even number, then the median of the list is the average of the middle two values.

Some integers are stored in a list variable my_list.  Please write some code to find and show the median of my_list in the console.

•my_list is guaranteed to be non-empty but my not be sorted.

•You should use integer division to calculate the average, if the size of the list is an even number.

•Please see the examples below to better understand the expected program behavior.

Example 1
Values in my_list [10, 2, 8, 4, 7]
Expected Program Output 7
Explanation After sorting, the list becomes [2, 4, 7, 8, 10].  The size of the list is 5 (odd number), so the median is just the middle value, which is 7.
Example 2
Values in my_list [10, 2, 20, 8, 120, 40, 800, 30]
Expected Program Output 25
Explanation After sorting, the list becomes [2, 8, 10, 20, 30, 40, 120, 800].  The size of the list is 8 (even number), so the median is the average of the middle two values (20 and 30), which is 25.

•Your can assume that the values stored in my_list are always valid.  You do not need to worry about invalid values or error handling.

•Your program will be tested (graded) using multiple sets of valid input values.  As long as the input values are valid, your program should always generate correct output values.

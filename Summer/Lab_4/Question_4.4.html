Some integers are stored in a list variable my_list.  Please write some code to test whether my_list is sorted or not.

•my_list could be empty, which is considered a sorted list.

•No matter my_list is sorted increasingly or decreasingly, it is considered a sorted list.

•Your code writes "sorted" in the console if my_list is sorted; "unsorted" otherwise.

•Please see the examples below to better understand the expected program behavior.

Example 1
Values in my_list [1, 1, 3, 5]
Expected Program Output "sorted"
Explanation The list is sorted non-decreasingly (increasingly).
Example 2
Values in my_list [9, 2, −1, −1, −10]
Expected Program Output "sorted"
Explanation The list is sorted non-increasingly (decreasingly).
Example 3
Values in my_list [3, 3, 3, 3]
Expected Program Output "sorted"
Explanation The list is sorted non-decreasingly and non-increasingly.
Example 4
Values in my_list [2, 6, 3, 1]
Expected Program Output "unsorted"
Explanation The list is not sorted.

•Your can assume that the values stored in my_list are always valid.  You do not need to worry about invalid values or error handling.

•Your program will be tested (graded) using multiple sets of valid input values.  As long as the input values are valid, your program should always generate correct output values.

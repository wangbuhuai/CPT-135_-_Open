Some integers are stored in a list variable my_list.

•Initially, you do not know the number of integers in my_list.

•my_list is guaranteed to have at least 2 integers with distinct values in it.

Please write some code to write the maximum value and strict second maximum value (in this order) in my_list in the console.

•The two output values should be separated by a single space in the console.

•Please see the examples below to better understand the expected program behavior and output format.

Example 1
Values in my_list [5, 7, −2, 9, 6]
Expected Output Values 9 7
Explanation The maximum value in the list is 9; the strict second maximum value in the list is 7.
Example 2
Values in my_list [6, 5, 6, 4]
Expected Output Values 6 5
Explanation The maximum value in the list is 6; the strict second maximum value in the list is 5.  Although there are two elements with value 6 in the list, the strict second maximum value should be strictly less than the maximum value.

•In this question, you can only use comparison operators; you cannot use max(), min(), append(), pop(), remove(), or count() functions or methods.

•Your can assume that the values stored in my_list are always valid.  You do not need to worry about invalid values or error handling.

•Your program will be tested (graded) using multiple sets of valid input values.  As long as the input values are valid, your program should always generate correct output values.

20		 m_1 = float("-inf")  # {m_1} is the maximum value.
m_2 = float("-inf")  # {m_2} is the strict second maximum value.

for i in range(len(my_list)):
    if my_list[i] > m_1:
        # If current value is greater than the current maximum value,
        # then both {m_1} and {m_2} need to be updated.
        m_2 = m_1  # The "old" maximum now becomes the strict second maximum.
        m_1 = my_list[i]
    elif my_list[i] == m_1:
        # Do nothing if current value is equal to the current maximum value,
        # since it cannot be the strict second maximum value.
        continue
    elif my_list[i] > m_2:
        # If current value is not greater than or equal to the current maximum,
        # but greater the current strict second maximum value,
        # then only {m_2} needs to be updated.
        m_2 = my_list[i]

print("%d %d" % (m_1, m_2))
